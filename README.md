# Darien Silversteen — Resume Site

Personal resume and project portfolio, hosted with GitHub Pages.

**Live site:** https://gabesnyder0101-code.github.io

## Pages

- `index.html` — resume (education, experience, school projects, extracurriculars, skills, awards)
- `projects.html` — project portfolio, linked from the resume's nav bar

## Updating

Edit the HTML files directly and push to `main` — GitHub Pages redeploys automatically within a minute or two.

To add a new project, see the instructions in the HTML comment at the top of the project cards in `projects.html`.
